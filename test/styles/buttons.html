Button modifiers

Any button can be modified to be slightly larger or have different width/height.

<button class="btn btn-large">
  A button
</button>

<button class="btn btn-main">
  A button
</button>

<button class="btn btn-primary btn-main">
  A button
</button>

Main buttons are used in toolbars, they are slightly larger then a regular button without the larger text size of a .btn-large button.
